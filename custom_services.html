Investigator report

The key starting point and our most recommended service for any serious endeavor to understand a new dataset. An investigator report is a single PDF describing the structure of the dataset, a collection of representative examples of individual samples or snapshots of the data, plots of all the basic statistics (e.g. histograms, tuning curves, timescale analysis, obvious correlations, basic dimensionality analyses, etc. for neural recordings), all the basic sanity checks, a brief summary of any obvious artifacts or unexpected features of the dataset. The report includes nothing state-of-the-art, but is a crucial foundation for more complex analyses, model fitting, and hypothesis testing.

Taking the time to write a report about a new dataset is one of the most important things I've learned in over a decade of analyzing complex, multi-faceted datasets in neuroscience. While it can be tempting to skip this step and jump straight into fitting a sophisticated model to your data, but if your goal is to transform data into interesting and reproducible scientific results, writing a report about all the basics in your data will make your life immensely easier. This is because it builds provides crucial intuition about the data, identifies all the features an expert would reasonably notice in looking over the dataset, serves as a critical reference point for more sophisticated analyses.

Along with the PDF of the report itself, this service will provide you with a version of your data that is easy to work with and code to reproduce all of the analyses in the report, and which can be easily built upon for more sophisticated analyses. (See our data analysis workflow for how we organize code and send it back to you).
