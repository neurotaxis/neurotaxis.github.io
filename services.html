Executive data report with code

A basic report on your data containing a collection of typical/atypical examples, summary of key artifacts, all the basic statistics, and miscellaneous comments about interesting phenomena. This provides a fundamental ground for beginning more sophisticated analyses and is crucial to developing a mental model of your data. An executive report also increases your data's half-life, making it infinitely easier to onboard new researchers. It is also allows you to interact with your dataset with an LLM, which will be crucial for searching data for phenomena of interest. We return all code used to make the report so that you can easily reproduce and extend it.
