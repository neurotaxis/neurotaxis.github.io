Package Offerings

Sand the floor

Organize the raw data and run all the obvious pre-processing steps. Write a basic report on the structure of the data, with many examples and views of raw and pre-processed data. The goal is to identify whether there is clear signal with interesting structure in the data, along with any obvious artifacts, estimating the complexity of the dataset, and to prepare the data for analysis. This is a crucial first step before proceeding further with a dataset. Sanding the floor is particularly useful when applied to multiple datasets, lending insight into which datasets are most likely to efficiently yield interesting scientific findings.

Paint the fence

Compute all the basic quantities and run all standard analyses. This includes making histograms of key variables, identifying obvious correlations, estimating dimensionality and timescales/nonstationarities, basic regressions, along with other clear analyses to do specific to a dataset. This is crucial for ensuring your data passes all the basic sanity checks and forms the foundation for more sophisticated analysis. This also forms the first pass of uncovering more subtle insights.
